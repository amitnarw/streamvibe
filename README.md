# StreamVibe – Premium OTT Streaming Platform

StreamVibe is a high-fidelity, dark-themed Over-The-Top (OTT) streaming website implementation. Built as a faithful recreation of a modern Figma UI design, it offers a premium user experience for discovering and watching movies and TV shows across multiple devices.

## 🎥 Project Overview

This project implements the flagship homepage of StreamVibe, focusing on a cinematic dark aesthetic, modular components, and responsive design.

### Key Sections:
- **Cinematic Hero**: A full-viewport background grid of movie posters with a centralized play interface and primary CTA.
- **Content Categories**: Interactive genre-based carousels with 2×2 thumbnail previews.
- **Device Ecosystem**: Informative cards showcasing platform availability (Smartphones, Tablets, Smart TV, Laptops, Consoles, VR).
- **Interactive FAQ**: A dual-column accordion for common user inquiries.
- **Subscription Plans**: Three-tier pricing comparison (Basic, Standard, Premium) with a Monthly/Yearly billing toggle.
- **CTA Banner**: High-impact "Start Free Trial" conversion section.
- **Omni-Footer**: Comprehensive site navigation, social connectivity, and legal disclaimers.

## 🚀 Getting Started

### Prerequisites
- Node.js (v18.x or later)
- pnpm (recommended) or npm

### Installation

1. Clone the repository:
   ```bash
   git clone [repository-url]
   cd streamvibe
   ```

2. Install dependencies:
   ```bash
   pnpm install
   ```

3. Run the development server:
   ```bash
   pnpm dev
   ```

4. Open [http://localhost:3000](http://localhost:3000) in your browser to view the site.
